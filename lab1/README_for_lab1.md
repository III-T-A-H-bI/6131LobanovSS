# Сравнение производительности суммирования векторов: CPU vs GPU

Этот Jupyter Notebook демонстрирует разницу в скорости выполнения операции суммирования векторов на центральном процессоре (CPU) и графическом процессоре (GPU) с использованием библиотеки CuPy.

## 📋 Описание

Код выполняет:
- Генерацию случайных векторов различной длины.
- Суммирование элементов вектора на CPU (чистый Python).
- Суммирование элементов вектора на GPU (CuPy).
- Сравнение времени выполнения и вычисление ускорения (speedup).
- Построение графика зависимости времени от размера вектора.
- Сохранение графика в файл `vector_sum_cpu_gpu.png`.

## 📊 Результаты

На выходе вы получите:
- Таблицу с временем выполнения (CPU и GPU) для каждого размера вектора.
- Ускорение (CPU time / GPU time).
- Среднее, максимальное и минимальное ускорение.
- График сравнения времени выполнения.

### Пример вывода (зависит от вашего оборудования):

| Vector Size | CPU (sec) | GPU (sec) | Speedup |
|-------------|-----------|-----------|---------|
| 1000        | 0.000043  | 0.000354  | 0.12    |
| 10000       | 0.000397  | 0.000095  | 4.17    |
| 50000       | 0.001861  | 0.000101  | 18.49   |
| ...         | ...       | ...       | ...     |

Среднее ускорение: **~48x**  
Максимальное ускорение: **~285x** (для вектора 1 000 000 элементов)
