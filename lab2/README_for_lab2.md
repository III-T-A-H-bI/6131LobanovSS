# Сравнение производительности умножения матриц: CPU vs GPU

Этот Jupyter Notebook демонстрирует разницу в скорости выполнения операции умножения матриц на центральном процессоре (CPU) и графическом процессоре (GPU) с использованием библиотеки CuPy.

## 📋 Описание

Код выполняет:
- Генерацию случайных квадратных матриц различного размера (от 100×100 до 2000×2000 с шагом 100).
- Умножение матриц на CPU (с помощью `numpy.dot`).
- Умножение матриц на GPU (с помощью `cupy.dot`).
- Сравнение времени выполнения и вычисление ускорения (speedup = время CPU / время GPU).
- Построение графика зависимости времени от размера матрицы.
- Сохранение графика в файл `cpu_vs_gpu.png`.


## 📊 Результаты

На выходе вы получите:
- Таблицу с временем выполнения (CPU и GPU) для каждого размера матрицы.
- Ускорение (CPU time / GPU time).
- Среднее, максимальное и минимальное ускорение.
- График сравнения времени выполнения.

### Пример вывода (зависит от вашего оборудования):

| Matrix Size | CPU (sec) | GPU (sec) | Speedup |
|-------------|-----------|-----------|---------|

Среднее ускорение: **~58x**  
Максимальное ускорение: **~145x**  
Минимальное ускорение: **1.87x**
